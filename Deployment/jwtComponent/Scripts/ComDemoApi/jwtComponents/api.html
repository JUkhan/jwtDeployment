grid properties
  1. options object type and it has the following properties
    1. limit int type
    2. loadingText string type
    3. filter bool type
    4. columns object type and it has the following properties
      1. field string type
      2. displayName string type
      3. sort bool type
      4. style onject type. ex: style:{width:'50px', color:'#333'}
      5. className string type
      6. render function type return string. ex: render:(row, inddex)=>'<b style="color:green">'+row.Name+'</b>'
      7. linkText string/string of array type ex: linkText:'Edit' or linkText:['Edit','Add']
      8. onClick function/function of array type ex: onClick:row=>{alert(row.Country)} or onClick:[row=>{alert(row.Country);},row=>{alert(row.Country);}]
  2. data object of array type
